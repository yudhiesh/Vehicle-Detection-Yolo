# Vehicle-Detection-Yolo

Vehicle Detection in C++ using YoloV3


## Dependencies to install
-> OpenCV

-> Build with OpenCV-Contribs

-> Install Darknet(Remember to set absolute file paths for coco.names, model cfg and weights cfg)
